# ReactCSV - Документация проекта

## Установка и запуск

### Установка

npm install

### Запуск проекта

Для разработки:
npm run start

Для сборки релиза:
npm run release

## Структура проекта

### Основные директории

```
src/
├── assets/        # Статические ресурсы (изображения, шрифты, стили)
├── components/    # React компоненты
├── constants/     # Константы и конфигурации
├── pages/         # Страницы приложения
├── php/          # PHP скрипты
├── store/        # Управление глобальным состоянием (Zustand)
├── App.jsx       # Корневой компонент
└── main.jsx      # Точка входа
```

### Подробное описание директорий

#### store/

Управление состоянием приложения через Zustand:

- **burgerStore.js** - Состояние мобильного меню
- **countryStore.js** - Управление данными о странах
- **ctaFormStore.js** - Состояние форм обратной связи
- **langStore.js** - Управление языковыми настройками

#### constants/

Конфигурационные файлы и константы:

- **site.config.jsx** - Основные настройки сайта
- **traffics.jsx** - Конфигурация трафика
- **copyrights.jsx** - Информация об авторских правах
- **forms.config.jsx** - Настройки форм

#### components/

Содержит все React компоненты, разделенные на категории:

- **layouts/** - Компоненты макета
- **functions/** - Вспомогательные функции
- **classes/** - Классы

#### assets/

Статические ресурсы проекта:

- Изображения
- Шрифты
- Стили
- Медиафайлы

#### php/

Серверные скрипты для обработки данных:

- **rssrequest.php** - Скрипт для получения RSS-новостей с investing.com
  - Поддерживает мультиязычность (ru, en, cn, de, es, it, fr)
  - Возвращает JSON с последними 6 новостями
  - Включает изображения, заголовки, авторов и даты публикации

- **feedback.php** - Обработчик обратной связи
  - Работает с SQLite базой данных (analytics.db)
  - Отслеживает уникальные IP-адреса пользователей
  - Генерирует уникальные номера для каждого пользователя
  - Использует механизм повторных попыток при блокировке базы данных

- **wallets.php** - Скрипт для работы с криптовалютными кошельками
  - Обработка и валидация кошельков
  - Интеграция с криптовалютными API

- **license.php** - Скрипт для проверки лицензий
  - Отправка email на почту

- **analytics.db** - SQLite база данных
  - Хранение данных
  - Отслеживание пользовательской активности

### Особенности структуры

- Модульная организация компонентов
- Разделение бизнес-логики и представления
- Централизованное управление состоянием
- Конфигурация через константы
- Поддержка мультиязычности
- Интеграция с PHP бэкендом

## Компоненты

### Основные компоненты

#### Layouts

Компоненты для организации структуры страниц:

- **docs/** - Компоненты для документации
- **mixins/** - Переиспользуемые компоненты или компоненты которые находятся на каждой странице
- **contacts/** - Компоненты для страницы контактов
- **invest/** - Компоненты для инвестиционного раздела
- **trading/** - Компоненты для торгового раздела
- **terminal/** - Компоненты терминала
- **404/** - Страница ошибки 404
- **collateral/** - Компоненты для работы с залогами
- **nav/** - Навигационные компоненты
- **index/** - Компоненты главной страницы
- **security/** - Компоненты безопасности
- **license/** - Компоненты лицензирования
- **about/** - Компоненты страницы "О нас"
- **tradingviewWidgets/** - Виджеты TradingView
- **forms/** - Компоненты форм
- **footer/** - Компоненты подвала сайта
- **news/** - Компоненты новостного раздела
- **popup/** - Модальные окна и попапы
- **inputs/** - Компоненты ввода данных

#### Functions

Функциональные компоненты для повторно используемой логики:

- **translate/** - Компоненты для интернационализации
- **isMobile.js** - Утилита для определения мобильных устройств
- **isLocalhost.js** - Утилита для определения локального окружения
- **beta/** - Компоненты для бета-функциональности

#### Classes

- **forms/** - Классовый компонент для работы с формами и запросами, содержит обьект с текстами ошибок

### Особенности компонентов

- Использование React Hooks для управления состоянием
- Интеграция с TradingView для отображения графиков
- Поддержка форм с валидацией через React Hook Form
- Модальные окна и попапы для интерактивности
- Адаптивный дизайн для всех компонентов
- Интеграция с API через Axios
- Управление состоянием через Zustand

## Зависимости

### Основные зависимости

#### UI и Компоненты

- **React** - Основная библиотека для построения пользовательского интерфейса
- **React DOM** - Пакет для работы с DOM в React
- **React Router DOM** - Маршрутизация и навигация в приложении
- **React Select** - Компонент выбора с поддержкой поиска и мультиселекта
- **React Slick** - Карусель и слайдер для React
- **React Phone Number Input** - Компонент ввода телефонных номеров с валидацией
- **React Auth Code Input** - Компонент для ввода кодов подтверждения
- **Air Datepicker** - Календарь и выбор даты
- **Lottie React** - Анимации в формате Lottie

#### Формы и Валидация

- **React Hook Form** - Управление формами с валидацией
- **React Helmet** - Управление мета-тегами и заголовками страниц

#### Работа с данными

- **Axios** - HTTP клиент для работы с API
- **Zustand** - Управление глобальным состоянием приложения
- **PapaParse** - Парсинг и работа с CSV файлами
- **Crypto JS** - Криптографические функции

#### Дополнительные компоненты

- **QRCode React** - Генерация QR-кодов

### Инструменты разработки

#### Сборка и оптимизация

- **Vite** - Современный сборщик для быстрой разработки
- **Gulp** - Автоматизация задач сборки
- **SASS** - Препроцессор CSS
- **FS Extra** - Расширенные возможности работы с файловой системой
- **Cross Spawn** - Кросс-платформенный запуск процессов

#### Линтинг и типизация

- **ESLint** - Линтер для JavaScript/React
- **ESLint Plugin React** - Правила линтинга для React
- **ESLint Plugin React Hooks** - Правила для React Hooks
- **ESLint Plugin React Refresh** - Поддержка Hot Module Replacement
